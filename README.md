# CipherStash Homebrew tap

Official Homebrew tap for CipherStash.

## Quickstart

To install the `stash`, the CipherStash CLI:

``` bash
brew install cipherstash/tap/stash
```

## Slowstart

Or to do the steps individually:

``` bash
# add the tap
brew tap cipherstash/tap

# install the cli
brew install stash
```
